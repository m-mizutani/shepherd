Searches messages and reads threads from the Slack workspace this Shepherd instance is connected to. Visibility is limited to channels the bot has been invited to (public channels by default, private channels only when explicitly added). Use `slack_search_messages` for keyword/modifier-based discovery (`from:@user`, `in:#channel`, `after:YYYY-MM-DD`), then drill into specific conversations with `slack_get_thread` or `slack_get_channel_history` to read full context. Resolve user IDs returned by other tools with `slack_get_user_info`.
